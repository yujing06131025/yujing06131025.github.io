About This Example

This HTML example includes some common elements you'll find on most webpages:

  • A `DOCTYPE` declaration for document type.
  • An `html` element as the root of the document.
  • A `head` element containing meta-information (like character set and viewport).
  • A `title` element which appears in the browser tab.
  • A `body` element containing the visible content of the page.
  • `h1` and `h2` heading elements for different levels of headings.
  • `p` elements for paragraphs of text.
  • `ul` (unordered list) and `li` (list item) elements for creating lists.
