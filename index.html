This page demonstrates how to present content in both English and Chinese.

About This Page (English)

This section is written primarily in English. We set the default language of the entire document to English using `<html lang="en">`. When specific parts of the content are in a different language, like Chinese in this example, we use the `lang` attribute on those specific elements.

This approach helps web browsers, screen readers, and search engines correctly interpret the language of different content blocks, improving accessibility and SEO.

关于本页面 (简体中文)

本节内容主要使用简体中文编写。我们将整个文档的默认语言设置为英语，即通过 `<html lang="en">` 实现。当内容的特定部分使用其他语言（如本例中的中文）时，我们会在这些特定的元素上使用 `lang` 属性。

这种方法有助于网络浏览器、屏幕阅读器和搜索引擎正确识别不同内容块的语言，从而提升可访问性和搜索引擎优化。
